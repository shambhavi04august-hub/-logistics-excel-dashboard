# 📦 Logistics & Supply Chain Dashboard (Excel Capstone Project)

## 📊 Overview

This capstone project presents an interactive Excel dashboard designed to analyze and visualize logistics and supply chain data. The dashboard provides insights into shipment performance, delivery efficiency, and operational trends.

## 🎯 Objectives

* Track and evaluate shipment performance 🚚
* Analyze on-time vs delayed deliveries ⏱️
* Identify inefficiencies in supply chain operations 📉
* Support data-driven decision-making 📊

## 📁 Dataset

The dataset includes:

* Order details
* Shipment information
* Delivery status
* Regional performance data

## 📊 Dashboard Features

* KPI Metrics (Total Orders, On-Time Deliveries, Delayed Shipments) 📊
* Interactive filters (Region, Category, Date) 🔍
* Visual charts for performance tracking 📈
* Clean and user-friendly dashboard layout

## 🛠 Tools & Techniques

* Microsoft Excel 💻
* Pivot Tables
* Data Cleaning
* Charts & Visualization

## 📈 Key Insights

* Identified regions with higher delivery delays
* Highlighted trends in shipment performance
* Improved visibility of operational efficiency
